# Intelligent Sensing Lab Coursework

This repository contains a university coursework web project for an Intelligent Sensing Lab website. The work is organised into three coursework iterations so reviewers can see the project evolve over time.

## Project Structure

- `CW1/` - first version of the multi-page static website.
- `CW2/` - second iteration with separated HTML, CSS, and image assets.
- `CW3/` - final iteration with HTML, CSS, JavaScript, images, and print styling.

## Final Version

The most complete version is in `CW3/`.

Open `CW3/HTML/Homepage.html` in a browser to view the site. The site includes:

- Home, About, Apply, Visit, Information, Contact, and Design pages.
- Responsive page-specific CSS files.
- JavaScript for the application and information pages.
- Image assets and print stylesheet support.

## Coursework Iterations

`CW1`, `CW2`, and `CW3` represent three rounds of improvement on the same project. They are kept as separate folders to make the development process and changes easy to review.
